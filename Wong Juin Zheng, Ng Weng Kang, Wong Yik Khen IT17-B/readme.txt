installation instruction----------------------------------------------

1. First please install the software in file "XAMPP Server installer".
2. Copy the GameTrade Code to the C:\xampp\htdocs
3. Import the database as the step below:
	1. go to the file directory of the XAMPP
	2. open the xampp_control.exe
	3. start both "Apache" and "MySQL", activate function can see by the background color that change to green after start.
4. Go to the website "http://localhost/phpmyadmin/"
5. click the "New" at the left hand side. then create a new database call "project".
5. Select the created database and click "Import",than select the sql file in file "Database SQL" and import it.
6. Go to http://localhost/GameTrade Code/ and enjoy.
